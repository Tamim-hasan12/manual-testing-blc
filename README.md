 Manual Testing Project (BLC Capstone)

 Overview
This project documents all Test Scenarios and Test Cases for a web application in an Excel sheet.  
It has been created primarily for learning **Manual Testing**.

Modules Covered
- Login  
- Dashboard  
- My Courses  
- Activities  
- Search  
- Profile & Logout  

Test Scenarios
Examples:
- TS-01: Verify student login with valid and invalid credentials  
- TS-04: Verify dashboard page load and welcome message  
- TS-09: Verify assignment and quiz visibility  
- TS-12: Verify profile menu access and logout functionality  

 Test Cases
Examples:
- TC-01: Login with valid student credentials → Expected: Redirect to dashboard  
- TC-02: Login with invalid password → Expected: Error message displayed  
- TC-07: Verify dashboard loads after successful login → Expected: Dashboard should load without error  
- TC-18: Search course by valid keyword → Expected: Matching courses should be displayed  
- TC-21: Logout from student account → Expected: User should be logged out successfully  

How to Use
1. Download the Excel file from this repository  
2. Open the sheet and review the **Test Scenarios** and **Test Cases**  
3. Update the **Actual Result** column during test execution
